# Portfolio

A React + Vite portfolio scaffold with a modern component structure and placeholder content.

## Available Scripts

- `npm install`
- `npm run dev`
- `npm run build`
- `npm run preview`

## Structure

This project includes:
- `src/components` for reusable UI and section components.
- `src/data` for project, skills, and social data.
- `public` for static assets.
