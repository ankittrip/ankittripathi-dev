## Local Setup

### Clone Repository

```bash
git clone https://github.com/ankittrip/ankittripathi-dev.git
cd ankittripathi-dev
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```

### Backend

```bash
cd backend
npm install
npm run dev
```
